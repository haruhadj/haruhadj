## 🤖 Agentic & Autonomous Workflows

Rather than using basic code-completion, I rely on a deeply integrated, CLI-driven environment where AI acts as an autonomous engineer operating on my terminal and editor.

* **Full-Codebase Navigation:** Harnessing **Claude Code** directly within Linux and VS Code's integrated workspace to map complex file relationships, execute sweeping multi-file refactors, and trace unfamiliar architectures instantly.
* **Autonomous Test-and-Fix Loops:** Offloading tedious test-suite tracking to my environment. When a lint or test error drops, my toolchain catches it, self-corrects the logic, and re-runs validations until green.
* **Deterministic Native Tooling:** Merging natural language with native Linux utilities. Claude Code drives `git`, handles branch creation, formats changes, and pushes production-ready pull requests directly from the terminal layer.
* **Structured Workspace Context:** Using project-level configuration files (`CLAUDE.md`) to define rigid type requirements, database schema boundaries, and linting guidelines so agentic actions always stay perfectly predictable.
